# 01. Posicionamiento del demo piloto

Esta versión no está pensada solo como demo visual.
Está pensada para presentarse a un cliente potencial como:
- simulación de carga
- entorno de evaluación
- evidencia comparativa
- piloto de prueba

## Objetivo
Mostrar que MitoPulse mejora:
- tasa de detección
- reducción de exposición
- reducción de falsos positivos
