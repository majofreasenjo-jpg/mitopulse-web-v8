# Physics & Resistance Layer

## FDE — Flow Dynamics Engine
Modela flujo, presión, velocidad, turbulencia y bottlenecks.

## FDI — Flow Distortion Index
Mide cuán anormal o distorsionado está el flujo.

## SSI — Systemic Stress-Strain Index
Mide deformación sistémica bajo presión.

## CFI — Cumulative Fatigue Index
Mide desgaste acumulado por micro-perturbaciones repetidas.

## FPI — Fracture Probability Index
Mide probabilidad de ruptura estructural.

## FEI — Free Energy Index
Mide energía libre útil para reorganización adaptativa.

## NEI — Non-Equilibrium Index
Mide distancia al equilibrio dinámico.

## KNI — Key Node Importance
Mide nodos cuya falla rompe el sistema.

## SRI — Structural Redundancy Index
Mide redundancia estructural y capacidad de respaldo.

## FGI — Field Gradient Index
Mide diferencias de presión/campo entre zonas del sistema.

## FSI — Feedback Stability Index
Mide estabilidad del sistema de feedback/control.

## ORI — Overshoot Response Index
Mide sobre-reacción del sistema ante perturbaciones.

## DTI — Delay Tension Index
Mide tensión creada por retrasos y desacoples temporales.
