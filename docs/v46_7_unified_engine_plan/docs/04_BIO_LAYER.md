# Bio Layer

## SEI — Systemic Energy Index
ATP-like indicator.
Mide energía operativa útil neta del sistema.

### Inputs
- Data Metabolism
- Reserve strength
- Homeostatic stability
- Regeneration capacity
- pressure load
- entropy drag

---

## SDI — Systemic Degeneration Index
Mide daño estructural y pérdida de integridad funcional.

### Inputs
- entropy
- reserve depletion
- regeneration failure
- identity drift
- predatory persistence

---

## PMI — Pathological Mutation Index
Mide mutación anómala persistente y orientada a evasión.

### Inputs
- mutation pressure
- adaptive change
- identity discontinuity
- repeated evasion success

---

## AES — Adaptive Evasion Score
Mide capacidad del sistema anómalo para esconderse y sobrevivir.

### Inputs
- camouflage
- persistence under suppression
- behavioral variability
- structural concealment

---

## SAI — Systemic Apoptosis Index
Mide cuándo conviene aislar o neutralizar un nodo/cluster para proteger el sistema.

---

## IML / IRI — Immune Memory / Immune Response
Miden memoria inmune y velocidad de respuesta del sistema.

---

## MSI — Metabolic Stress Index
Mide carga operativa vs capacidad metabólica del sistema.

---

## SII — Signal Integrity Index
Mide integridad de la señal y ruido / manipulación.
