# Flujo de Onboarding Técnico

## Fase 1 — Discovery técnico
- identificar sistemas fuente
- identificar protocolo / formato
- identificar frecuencia
- identificar owner de datos
- definir tenant
- definir vertical y objetivos

## Fase 2 — Mapping
- mapear objetos del cliente a modelo canónico
- definir entidades críticas
- definir relaciones
- mapear eventos
- mapear campos externos

## Fase 3 — Ingesta mínima
- cargar sample entities
- cargar sample events
- validar timestamps
- validar tenant isolation
- validar baseline

## Fase 4 — Overlay
- activar dashboard
- activar índices globales
- activar índices sectoriales generados por IGE
- definir outputs por audiencia

## Fase 5 — Acción / gobernanza
- activar sandbox actions
- definir destinatarios de alertas
- validar explainability
- validar audit log
