# Minimum Data Contract

## Campos mínimos obligatorios
- entity_id
- entity_type
- timestamp
- event_type
- value / amount / metric
- source_system

## Campos recomendados
- relationship_from
- relationship_to
- severity
- location / region
- tenant_id
- outcome_label

## Por vertical

### Marketplace
- seller / buyer / device / payout / listing / review velocity

### Bank
- account / wallet / beneficiary / device / transfer / branch

### Energy
- supplier / unit / tank / route / contract / external field
