# Pilot Success Criteria

## Cuantitativos
- 3–5 hallazgos no visibles previamente
- reducción de tiempo de interpretación
- priorización de alertas mejorada
- reducción de ruido / falsos positivos

## Cualitativos
- el cliente entiende por qué el sistema actúa
- el dashboard separa bien capas ejecutivas y técnicas
- el piloto demuestra valor económico u operacional

## KPIs sugeridos
- alert_precision_proxy
- intervention_speed_proxy
- hidden_pattern_discovery_count
- avoided_loss_estimate
