# Onboarding Checklist

## Negocio
- sponsor ejecutivo
- owner técnico
- owner de riesgo / operaciones
- unidad receptora de alertas

## Datos mínimos
- entidades
- eventos
- timestamps
- relaciones / dependencias
- labels históricos si existen

## Operación
- tenant creado
- credenciales iniciales
- modo de cliente activo
- conectores externos necesarios definidos

## Seguridad
- sandbox habilitado
- auditoría habilitada
- retención de logs definida
