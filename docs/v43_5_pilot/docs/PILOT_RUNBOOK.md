# Pilot Runbook (4–6 semanas)

## Semana 0 — Discovery
- definir sponsor
- identificar unidad receptora de alertas
- mapear fuentes de datos
- seleccionar vertical: marketplace / bank / energy

## Semana 1 — Ingesta
- validar esquema
- normalización semántica
- activar Semantic Ingestion AI
- cargar dataset histórico + corriente

## Semana 2 — Baseline
- calcular baseline de NHI / TPI / SCR / MDI
- definir thresholds iniciales
- validar outputs con cliente

## Semana 3 — Hallazgos
- entregar 3–5 hallazgos
- mostrar playback
- revisar falsos positivos
- ajustar behavioral profiles

## Semana 4 — Acción
- activar sandbox actions
- probar flujo alerta → decisión → acción
- confirmar destinatarios reales

## Semana 5–6 — Cierre
- reporte ejecutivo
- reporte técnico
- ROI / riesgo evitado
- recomendación de siguiente fase
