# Scientific Benchmark Engine

## Meta
Subir de 4 a 9+.

## Qué hace
Compara MitoPulse contra baselines por dominio.

## Baselines mínimos
- Logistic Regression
- Random Forest
- XGBoost
- score/rule baseline del cliente cuando exista

## Qué se compara
- ROC AUC
- PR AUC
- top-k precision
- false positive cost
- false negative cost
- lead-time gain
- utility / avoided loss proxy

## Entregables
- benchmark pack por dominio
- quarterly scorecards
- reproducible evaluation config
- executive benchmark summary
