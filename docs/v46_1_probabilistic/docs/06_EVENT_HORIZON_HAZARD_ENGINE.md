# Event-Horizon / Hazard Engine (EHE)

## Objetivo
Pasar de "riesgo alto" a "qué tan probable es que ocurra y cuándo".

## Qué calcula
- hazard_score
- time_to_criticality
- critical_window_probability
- regime_transition_probability

## Salidas
- probability_of_critical_event_next_window
- expected_time_to_criticality
- urgency_grade

## Interpretación
No solo medimos severidad.
Medimos cercanía temporal al umbral crítico.
