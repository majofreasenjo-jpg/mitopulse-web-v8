# Structured Uncertainty Engine (SUE)

## Objetivo
No devolver un solo número, sino una probabilidad con contexto de confianza.

## Tipos de incertidumbre
- data scarcity uncertainty
- noise / aleatoric uncertainty
- regime uncertainty
- incomplete observation uncertainty

## Outputs
- p_mean
- p_low
- p_high
- uncertainty_bandwidth
- confidence_grade
- uncertainty_source_mix

## Ejemplo
SCR_calibrated = 0.74
interval = [0.61, 0.83]
confidence = medium-high
