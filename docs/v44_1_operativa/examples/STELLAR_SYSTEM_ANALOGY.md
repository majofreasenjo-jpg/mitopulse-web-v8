# Stellar / Planetary Analogy

## Mapping
- gravitational fields -> relational gravity / external field
- micro-perturbations -> pulses / seismic-like precursors
- orbital stability -> homeostasis / attractor regime
- collapse / flare -> criticality / SCR / trigger zone

This does not mean astrophysics numerically out of the box.
It means MitoPulse can model stellar systems as dynamic ecosystems of pressure, coupling and collapse.
