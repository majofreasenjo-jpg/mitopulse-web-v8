# Deploy Guide

## Local
1. copiar `.env.example` a `.env`
2. instalar dependencias
3. correr backend
4. abrir dashboard

## Render
1. subir repo
2. conectar `render.yaml`
3. cargar variables de entorno
4. validar login / tenants / audit logs

## Recomendación
Para piloto usar:
- sandbox actions = true
- audit logging = true
- AI explainability = true
