# MitoPulse v43.5 Pilot Package

Paquete de piloto comercial y operativo para ejecutar un piloto de 4–6 semanas.

## Objetivo
Reducir la fricción entre demo y cliente real con:
- onboarding
- definición de datos mínimos
- KPIs de éxito
- plan semanal
- outputs para cliente
- hardening mínimo de operación

## Contenido
- runbook del piloto
- checklist de onboarding
- contrato de datos mínimo
- plantillas por vertical
- criterios de éxito
- demo guiada
- paquete de despliegue local / Render
