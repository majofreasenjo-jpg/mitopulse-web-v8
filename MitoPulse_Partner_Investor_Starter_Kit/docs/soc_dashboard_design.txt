
MITOPULSE — SOC DASHBOARD DESIGN

The MVP dashboard should resemble a cybersecurity platform.

Main panels:

1) Network Graph
Real-time visualization of nodes and relationships.

2) Fraud Spread Map
Shows how risk propagates through the graph.

3) Behavior Monitor
Displays learned routines and anomalies.

4) Guardian Decisions
Institutional oversight actions.

5) Attack Simulation Lab
Sandbox to run fraud scenarios.

Design inspiration:
- Darktrace
- CrowdStrike
- Cloudflare dashboards

Color palette:
dark background
neon network lines
alert colors for risk propagation
