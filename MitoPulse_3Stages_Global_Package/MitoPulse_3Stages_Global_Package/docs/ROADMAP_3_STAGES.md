# Roadmap de 3 etapas

## Etapa 1
Demo comercial sólida.

## Etapa 2
Hardening del core:
- multitenancy
- auth
- billing
- trust graph
- ledger

## Etapa 3
Clientes globales y operación online:
- PWA instalable
- wrapper móvil
- scaffolds wearable
- HTTPS / reverse proxy / tunnel

## Nota
No existe un único binario que se instale literalmente en todos los teléfonos y todos los wearables.
Este paquete deja una base real para:
- PWA instalable en muchos móviles
- wrapper móvil Android/iOS
- scaffolds wearable
- operación online
