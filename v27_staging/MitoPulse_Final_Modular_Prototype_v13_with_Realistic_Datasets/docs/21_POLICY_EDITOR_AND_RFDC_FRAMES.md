# Policy Editor + RFDC Graph Frames

Esta versión cierra dos mejoras operativas:

## 1. Policy Editor en la UI
Permite:
- cargar políticas actuales
- editar JSON por cliente
- guardar playbooks sin tocar código

## 2. Playback con frames RFDC
El timeline ahora usa:
- snapshots del grafo RFDC
- enlaces inferidos
- ondas relacionales
- métricas por frame
- action threshold y acción recomendada
