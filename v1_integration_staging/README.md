# MitoPulse Integration Architecture Pack v1

Paquete de arquitectura de integración para conectar MitoPulse como capa superior
sobre sistemas existentes en banca, marketplace, energía, logística y entornos ERP/administrativos.

## Incluye
- mapa de conectores por industria
- modelo canónico
- flujo de onboarding técnico
- patrones de overlay sobre sistemas existentes
- integración con ERP / SAP-like
- demos de integración (no solo dataset)
- guías para mostrar facilidad de adopción al cliente

## Principio rector
MitoPulse no reemplaza el stack actual del cliente.
MitoPulse se integra como overlay de inteligencia sistémica.
