# MitoPulse v46.3 — Master Dashboard Integrated

Integración del módulo Live Graph + Forecast + WaveEngine + Invisible Storm
dentro del dashboard maestro del sistema operativo MitoPulse.

## Tabs incluidos
- Executive
- Graph Live
- Forecast
- System Brain
- AI Layer
- Simulation
- Invisible Storm Improved
- Alerts & Actions
- MitoPulse Verify

## Cómo correr
1. `pip install -r requirements.txt`
2. `uvicorn backend.main:app --reload`
3. abrir `http://127.0.0.1:8000`
