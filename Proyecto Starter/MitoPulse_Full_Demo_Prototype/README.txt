MitoPulse Full Demo Prototype

Qué es
- Demo ejecutable más completa para ofrecer el concepto MitoPulse.
- Incluye:
  1) Human Check
  2) Dashboard
  3) Simulador de sesiones humanas y bots
  4) Flujos demo de uso

Cómo ejecutar
1. Instala Python 3.9 o superior
2. Abre terminal en esta carpeta
3. Ejecuta:

   python server.py

4. Abre en tu navegador:

   http://localhost:8080

Qué puedes probar
- Inicio: demo principal
- Human Check: verificación humana manual
- Dashboard: métricas y eventos
- Simulator: genera humanos y bots de prueba

Notas
- Es un prototipo conceptual.
- No usa base de datos ni seguridad de producción.
- Sirve para demos, pruebas y conversaciones con socios o potenciales clientes.
