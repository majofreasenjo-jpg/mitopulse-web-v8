# Extra scope notes

## Analogía sísmica
Sí, MitoPulse ya puede modelar presión acumulativa tipo sísmica:
- micro perturbaciones
- acumulación de presión
- criticalidad
- ruptura / trigger

## Crimen organizado
Sí, MitoPulse es muy adecuado para:
- coordinación oculta
- células desconectadas
- sincronía conductual
- presión y propagación de redes criminales

## Posicionamiento correcto
No venderlo como sistema ofensivo.
Sí venderlo como:
- inteligencia sistémica
- anticipación de amenazas
- detección de coordinación oculta
- defensa adaptativa
