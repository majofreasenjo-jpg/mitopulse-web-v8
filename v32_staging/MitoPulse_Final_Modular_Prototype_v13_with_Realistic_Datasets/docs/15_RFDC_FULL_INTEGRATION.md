# RFDC Full Integration

Esta versión integra los 4 módulos faltantes en el motor central:

- Relational Dark Matter + MDI
- Relational Wave Engine
- Fraud Evolution Engine
- Guardian Swarm

## Núcleo nuevo
- `core/rfdc.py` → Relational Field Dynamics Core

## Endpoint nuevo
- `/api/rfdc/run`

## Qué hace RFDC
1. corre detección de anomalías
2. calcula MDI
3. infiere clusters ocultos
4. resume ondas relacionales
5. calcula NHI / TPI / SCR
6. genera patrón evolutivo de fraude
7. valida alertas con Guardian Swarm
