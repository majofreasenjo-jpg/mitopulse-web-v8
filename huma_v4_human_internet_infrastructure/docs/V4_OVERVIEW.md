# Huma v4 Overview

## Nuevos módulos
- edge_orchestrator
- identity_continuity_protocol
- reputation_network

## Qué agregan
### edge_orchestrator
Normaliza la entrada y prepara payloads para el gateway.

### identity_continuity_protocol
Mide continuidad operativa entre sesiones, dispositivos y trayectoria histórica.

### reputation_network
Mantiene una reputación simplificada por tenant y usuario, como base de una trust network más grande.
