# Android + Wear OS (Scaffold)

Este directorio contiene **código fuente mínimo** (Kotlin) para:
- Android (teléfono)
- Wear OS (módulo stub)

⚠️ Para compilar realmente:
- Crea un proyecto en Android Studio (Empty Activity)
- Copia los archivos de `src_snippets/` a tu proyecto real.
- Ajusta Gradle: Retrofit/OkHttp, coroutines, etc.

Se entrega como scaffold porque el proyecto Android completo (Gradle wrapper, etc.) varía según versión de Android Studio.
