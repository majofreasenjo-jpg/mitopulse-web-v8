# MitoPulse v68 — Module 3 Benchmark Engine

Motor de benchmark para demostrar superioridad vs baseline.

## Qué incluye
- Métricas: Lead Time, Precision, FPR/FNR
- Estimación de Loss Prevented
- System Advantage Index (SAI)
- Generación de reportes comparativos

## Instalación
pip install -r requirements.txt

## Demo
python scripts/run_benchmark_demo.py
