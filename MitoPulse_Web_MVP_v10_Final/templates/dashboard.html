Qué demuestra este MVP

1. Detección de fraude humano y social.

2. Red de confianza con nodos persistentes.

3. Guardianes que estabilizan la red.

4. Mapa de campañas y nodos infecciosos.

5. Métricas explícitas de falsos positivos y falsos negativos.

Arquitectura operativa

Usuarios → Cloudflare/WAF → API Gateway → MitoPulse Core → Trust Graph → Shared Reality → Fraud Hunter → Decision Engine.

Integraciones externas sugeridas: Cloudflare, Arkose, BioCatch/Featurespace como señal adicional, Chainalysis/TRM para crypto.
