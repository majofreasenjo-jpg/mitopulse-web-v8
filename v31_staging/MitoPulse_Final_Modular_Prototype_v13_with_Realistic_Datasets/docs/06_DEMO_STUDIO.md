# Demo Studio

## Qué debe verse en pantalla grande
1. perfil del cliente
2. módulos activos
3. estado del ecosistema
4. lógica visible de lo que está actuando
5. nodos / clusters / coordinación
6. señales e índices
7. conclusión del caso

## Narrativa ideal de 7 minutos
- contexto del cliente
- estado normal
- inicio de amenaza
- coordinación / gravedad / cluster
- detección de MitoPulse
- comparación baseline vs MitoPulse
- impacto esperado
