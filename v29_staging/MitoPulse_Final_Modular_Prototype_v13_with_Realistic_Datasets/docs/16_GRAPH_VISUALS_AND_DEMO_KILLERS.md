# Graph Visuals + Demo Killers Integration

Esta versión agrega dos capas visuales clave:

## 1. Visualización sobre el grafo
Endpoint:
- `/api/rfdc/graph`

Muestra:
- nodos alerta
- clusters ocultos
- enlaces inferidos (líneas punteadas)
- halos de ondas relacionales
- colores por tipo de entidad/riesgo

## 2. Demo Killers conectadas a RFDC
Endpoint:
- `/api/demo/run/<demo_id>`

Demos disponibles:
- `invisible_network`
- `invisible_storm`
- `coming_collapse`

## Dashboard
Se agregan:
- `Graph + Waves + Dark Matter`
- `Demo Killers Studio`
