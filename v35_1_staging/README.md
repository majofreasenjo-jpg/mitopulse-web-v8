# MitoPulse v35.1 FULL UI REAL

Versión amplia, modular y presentable de MitoPulse para demos, simulación y operación.

## Incluye
- Executive View
- System Brain
- System Dynamics
- Simulation & Demo Killers
- Reports & Alerts
- Control Center
- Yahoo / Binance / Macro mock feeds
- Sandbox action flow
- Demo killers con play/pause

## Ejecutar
1. `pip install -r requirements.txt`
2. `uvicorn app:app --reload`
3. Abrir `http://127.0.0.1:8000`

## Qué muestra
- KPIs gruesos para cliente
- pestaña técnica con motores y cálculo
- pestaña de dinámica con waves / field / triggers
- pestaña de reportes con alerta enviada al cliente/unidad especializada
- pestaña de simulación con los 3 demo killers
